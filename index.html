Google Calendar
To add automatic OC-AAN updates to Google Calendar:
  1. Click “Copy Google URL” below
  2. Open Google Calendar on a computer
  3. On the left side, click the + next to “Other calendars”
  4. Select From URL
  5. Paste the copied URL and click Add calendar
Note: Google Calendar subscriptions must be added from a computer browser, not the mobile app.

Outlook on the web / New Outlook
Paste the calendar URL below into Calendar > Add calendar > Subscribe from web and include the full https:// address.
  1. Copy the URL
  2. Open Outlook on the web or New Outlook
  3. Choose Subscribe from web
  4. Paste the full URL and save

Outlook Classic
Outlook Classic may not open this feed from the prompt dialog. For the best experience, use Outlook on the web or New Outlook. If you need Outlook Classic, follow these steps:
  1. Open Outlook Classic
  2. Go to Calendar > Add Calendar > From Internet
  3. Paste the full URL below, including https://
  4. Save / Subscribe
