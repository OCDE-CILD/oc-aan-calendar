Outlook Classic
Outlook Classic may not open this feed from the prompt dialog. For the best experience, use Outlook (new) or Outlook on the web. If you need Outlook Classic, follow these steps:
  1. Open Outlook Classic
  2. Go to Calendar > Add Calendar > From Internet
  3. Paste the full URL below, including https://
  4. Save / Subscribe
https://ocde-cild.github.io/oc-aan-calendar/oc-aan.ics
Download Static ICS File
Recommended: Use the live calendar option to receive automatic meeting updates, schedule changes, and revised event details.

The “Download Static ICS File” option creates a one-time copy of the calendar and will not update automatically.

Compatible with Outlook, Google Calendar, Apple Calendar, and mobile devices.
