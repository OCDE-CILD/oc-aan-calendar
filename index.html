Open in Apple / Mobile Calendar
For Apple Calendar, iPhone, iPad, and some mobile calendar apps.
Google Calendar
To receive automatic OC-AAN calendar updates in Google Calendar:
  1. Click Copy Google URL below
  2. Open Google Calendar on a computer
  3. On the left side, click the + next to “Other calendars”
  4. Select From URL
  5. Paste the copied URL
  6. Click Add calendar
Note: Google Calendar subscriptions must be added from a computer browser, not the mobile app.
https://ocde-cild.github.io/oc-aan-calendar/oc-aan.ics
Outlook
To receive automatic OC-AAN calendar updates in Outlook:
  1. Click Copy Outlook URL below
  2. Open Outlook
  3. Select one of these options:
    • Subscribe from web in Outlook on the web or New Outlook
    • From Internet in Outlook Classic
  4. Paste the copied URL, including https://
  5. Save / Subscribe
Recommended: add the feed in Outlook on the web or New Outlook first. If you also use Outlook Classic with the same account, the subscription should appear there after Outlook syncs.
https://ocde-cild.github.io/oc-aan-calendar/oc-aan.ics
Download Static ICS File
Recommended: Use the live calendar options above to receive automatic meeting updates, schedule changes, and revised event details.

The “Download Static ICS File” option creates a one-time copy of the calendar and will not update automatically.

Compatible with Outlook, Google Calendar, Apple Calendar, and mobile devices.
